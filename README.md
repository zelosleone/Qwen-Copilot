Use Qwen Code models with VS Code Copilot's built-in agent experience.

- Access to Qwen Code models (qwen3-coder-plus, qwen3-coder-flash)
- OAuth sign-in, don't bother with tokens. It's free anyways!
- Native tool calling

## Get started

1. Install the extension.
2. Open the Command Palette and run `Qwen Copilot: Authenticate`.
3. Start chatting in VS Code and pick a Qwen model.

## Requirements

- VS Code 1.96.0 or later
- A Qwen account at https://chat.qwen.ai
